@attrus-ds/core v0.2.0 Jun 2026

The design system that powers ATTRUS.

A token-driven foundation for ATTRUS — the financial-operations platform for Latin American treasuries. Every colour, type ramp, spacing step, radius, shadow, and component lives here, in one canonical npm package. Change a token; every screen re-skins.
