Changelog v1.0.0

Every notable change to the design system, grouped by release. Versioning follows SemVer; the version number lives in version.js and is mirrored across the catalog and index pages.

1.0.0

Current
2026-06-19 · first tagged release

Added

  • Multi-product theming — two orthogonal axes: data-theme (light/dark) × data-product (Enterprise / Business). Theming surface spans color, gradients, inverse layer and shape.
  • Product themes — themes/enterprise.css (Navy) + themes/business.css (Blue), loaded via a single themes/index.css; new product = 1 file + 1 import + 2 registrations. themes/_template.css scaffold included.
  • Product & theme controls — selector + dark-mode toggle in the catalog, app topbar, and both index hubs; persisted, anti-FOUC (theme-toggle.js).
  • TypeScript layer — 20 typed component wrappers (components/*/*.tsx) over the canonical CSS, each with a live bench gallery.
  • Dark mode — full token remap via [data-theme="dark"]; validated across app, catalog and docs.
  • Data-viz tokens, token export (JSON / Style Dictionary), and the drift linter (tools/drift-linter.html) guarding CSS↔TSX↔theming.
  • Index hubs — preview/index.html (example pages) and components/index.html (TSX benches), cross-linked; bench pages get a back-link.
  • This changelog + version.js single-source version, mirrored on the catalog and index pages.

Changed

  • Brand surface tokenized — every component (CSS + TSX + app) consumes --color-brand-* / --color-accent-* instead of raw scales, so a product reskins the whole system with no component edits.
  • Gradients & inverse layer derive from --color-brand-solid — hero backdrops and dark/premium panels follow the active product.
  • Shape tokens (--shape-control/-container/-overlay) pick steps from the radius scale; Business reads rounder than Enterprise.

Fixed

  • Dark-mode contrast — skeletons, the Failed table row, and ~52 raw light-tint usages across components and docs migrated to theme-aware tokens.
  • Stray Royal refs — sidebar active count, combobox chips, file-action hover and focus borders now product-aware.
  • Cache-busting on theme-toggle.js / themes/index.css to avoid stale tabs.
