Color

03Color · semantic roles

Components never reach for raw palette values. They consume semantic roles named by purpose — --color-background-canvas, --color-foreground-default, --color-border-focus. Each role has a light variant and an inverse twin so dark surfaces work without forks.

Primitives — absolute anchors used by the semantic layer; never consumed directly by components

--color-pure-whiteAbsolute white — neutral-0 anchor
--color-pure-blackAbsolute black — for logo / print only
--color-shadow-tintDefault ink for box-shadow color-mix
--color-scrimModal / drawer backdrop

Ink tones — inverse-surface variants used by sidebar, hero, dark popovers

--color-ink-baseDefault inverse surface
--color-ink-raisedRaised cards on inverse
--color-ink-deepDeepest sunken — wells, recessed
--color-ink-hoverHover layer on inverse
