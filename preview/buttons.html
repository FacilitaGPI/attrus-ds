Design system

Buttons

Pill-shaped, action-driving controls. Triggers a single, immediate action — submit a form, confirm a destructive change, navigate to a flow. For navigation that should look passive (in-page anchors, footer links), use a plain link.

01Anatomy

A button is built from three optional parts inside a pill container. Order is leading icon → label → trailing icon; never break this order. Icons are optional but the label is mandatory unless the button is in .btn-icon mode.

1 Leading icon Reinforces the action's meaning before the label is read.
2 Label Action verb in sentence case. 1–3 words.
3 Trailing icon Hints at progression — arrow, chevron, external link.
4 Pill container radius-pill · ring on focus · 24/32/40/48px tall.

05Composition

Buttons rarely live alone. Three established patterns: action pairs (form footer), toolbars (mixed ghost/secondary), and segmented groups (one logical control split into mutually-exclusive options).

Form footer

Right-aligned. Cancel as Ghost on the left, Primary on the right. Never reverse the order.

Destructive confirmation

Cancel as Ghost. Destructive action solid, focus-ring tinted danger.

Toolbar

Ghost icon-only buttons grouped tightly with secondary text actions for context.

Segmented group .btn-group

One logical control with mutually-exclusive options. Use Secondary as base.

07Don'ts

Common pitfalls. Each undermines the visual hierarchy or the accessibility contract the component depends on.

Two Primaries side-by-sideThe user can't tell which is the recommended path. Demote one to Secondary or Ghost.

Danger for non-destructive actionsRed is a strong signal — reserve it for irreversible deletions. Save is Primary.

Square cornersAll buttons in this system are pill-shaped. Don't override --radius-pill.

ALL CAPS labelsSentence case only. ALL CAPS is reserved for overlines and labels — not interactive controls.

Hit areas under 24pxBelow the XS minimum, buttons fail touch targets. Always pick a defined size.

Two trailing-meaningful iconsPick one — leading reinforces meaning, trailing hints at progression. Never both.
